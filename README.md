# Monthly Work Schedule Planner

Simple PHP + JavaScript app for planning work days by month.

## Features
- Month picker for any month.
- Calendar view with one cell per day.
- Set each day as:
  - Full day
  - Half day
  - Day off
- Data persists in `data/schedule.json`.

## Run locally
```bash
php -S 0.0.0.0:8000
```
Then open `http://localhost:8000`.
